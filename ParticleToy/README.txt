The particles react to anchor points on the screen. They change behavior
depending on the number of anchors.

You can place anchors by holding [Ctrl] and using the left mouse button.
Pressing the left mouse button without holding [Ctrl] will erase all
anchors and place one single new one.

You can also place a number of random anchors using the
[F1-F12] Keys. [F1] will place one, [F2] will place two and so on.

Holding [Ctrl] will also pause the animation and show all anchors and a
number of how much there are.

More controls are:
[Ctrl]+[F11] toggle fullscreen
[Esc] toggle info
[R] start/stop frame recording
